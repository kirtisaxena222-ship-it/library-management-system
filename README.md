# Library Management System (C++)

A console-based Library Management System built using C++ and OOP concepts.

## Features
- Add Books
- Display All Books
- Issue Book
- Return Book
- Menu Driven System

## Concepts Used
- Classes & Objects
- Vectors (Dynamic Storage)
- Encapsulation
- Conditional Statements
- Loops

## How to Run

Compile:
g++ main.cpp -o library

Run:
./library
